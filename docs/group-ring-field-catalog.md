# Group, Ring, and Field API Catalog

This document catalogs the public Python-facing functionality in the algebraic packages exposed by `algebrapy`, grouped by module and class. The descriptions are intentionally short and operational.

Unless otherwise noted, polynomial-style coefficient lists are written from low degree to high degree:
- `[a0, a1, a2, a3]` means `a0 + a1*x + a2*x^2 + a3*x^3`

## Group package

### `group.perm.Perm`

Represents a permutation of `{0, ..., n-1}` in image form.

#### Construction

- `Perm(n, images)`:
  Create a permutation from its images. `images` must be a bijection on `0..n-1`.
- `Perm.identity(n)`:
  Return the identity permutation on `n` points.
- `Perm.cycle(n, cycle)`:
  Build a permutation from cycle notation written as a list such as `[0, 1, 2]` for `(0 1 2)`. Short cycles of length `< 2` return the identity.

#### Accessors and core methods

- `n()`:
  Return the permutation degree.
- `as_images()`:
  Return the underlying image list.
- `compose(other)`:
  Return `self ∘ other`, meaning apply `other` first and then `self`.
- `inv()`:
  Return the inverse permutation.
- `pow(exp)`:
  Raise the permutation to an integer power. Negative powers use the inverse; `0` returns the identity.
- `order()`:
  Return the group-theoretic order of the permutation, computed as the least common multiple of its cycle lengths.

#### Python protocol methods

- `__mul__(other)`:
  Alias for composition, so `p * q` means `p.compose(q)`.
- `__repr__()`:
  Return a debug-style representation.

### `group.perm.Sn`

Represents the symmetric group `S_n` with small-scale enumeration and subgroup generation utilities.

#### Construction

- `Sn(n)`:
  Construct `S_n`. The implementation restricts `n <= 8` to keep brute-force enumeration manageable.

#### Accessors and core methods

- `n()`:
  Return the degree `n`.
- `identity()`:
  Return the identity permutation in `S_n`.
- `elements(max_size=None)`:
  Enumerate all permutations in `S_n`, failing if the result would exceed `max_size`.
- `generated(gens)`:
  Compute the subgroup generated by `gens` using the default size bound.
- `generated_with_limit(gens, max_size)`:
  Compute the subgroup generated by `gens`, including inverses, and fail if the generated subgroup exceeds `max_size`.

#### Python protocol methods

- `__repr__()`:
  Return a short representation such as `S4`.

## Ring package

### `ring.zn.Zn`

Represents the residue ring `Z/nZ`.

#### Construction

- `Zn(n)`:
  Construct the ring modulo `n`. Requires `n >= 2`.

#### Accessors and constructors

- `modulus()`:
  Return the modulus `n`.
- `zero()`:
  Return the additive identity.
- `one()`:
  Return the multiplicative identity.
- `elem(k)`:
  Convert an integer to its canonical residue class modulo `n`.
- `elements()`:
  Enumerate all residue classes.
- `units()`:
  Enumerate all invertible elements.
- `zero_divisors()`:
  Enumerate all nonzero zero divisors.

#### Ring operations

- `is_unit(a)`:
  Return whether `a` is invertible in `Z/nZ`.
- `is_zero_divisor(a)`:
  Return whether `a` is a nonzero zero divisor in `Z/nZ`.
- `is_integral_domain()`:
  Return whether `Z/nZ` is an integral domain.
- `add(a, b)`:
  Return `a + b`.
- `neg(a)`:
  Return `-a`.
- `sub(a, b)`:
  Return `a - b`.
- `mul(a, b)`:
  Return `a * b`.
- `inv(a)`:
  Return the multiplicative inverse of `a` if it exists. Fails for `0` and non-units.
- `pow(a, e)`:
  Return `a^e`. Negative exponents require `a` to be a unit.

### `ring.zn.ZnElem`

Represents an element of a fixed ring `Z/nZ`.

#### Element methods

- `value()`:
  Return the canonical integer representative in `0..n-1`.
- `is_zero()`:
  Return whether the element is zero.
- `is_unit()`:
  Return whether the element is invertible.
- `is_zero_divisor()`:
  Return whether the element is a nonzero zero divisor.
- `inv()`:
  Return the multiplicative inverse if it exists.

#### Python protocol methods

- `__bool__()`:
  Treat nonzero elements as truthy.
- `__int__()`:
  Convert to the canonical integer representative.
- `__repr__()`:
  Return a string such as `7 (mod 12)`.
- `__richcmp__(other, op)`:
  Support `==` and `!=` for elements from the same parent ring.
- `__neg__()`:
  Return the additive inverse.
- `__add__(other)`:
  Support element addition.
- `__sub__(other)`:
  Support element subtraction.
- `__mul__(other)`:
  Support element multiplication.
- `__truediv__(other)`:
  Divide by `other` using multiplicative inversion. Requires `other` to be a unit.
- `__pow__(exp, modulo=None)`:
  Support `a ** exp` and `pow(a, exp)`. Three-argument `pow` is rejected.
- `__radd__(other)`:
  Support `int + ZnElem`.
- `__rsub__(other)`:
  Support `int - ZnElem`.
- `__rmul__(other)`:
  Support `int * ZnElem`.
- `__rtruediv__(other)`:
  Support `int / ZnElem`, interpreted inside `Z/nZ`.

## Field package

### `field.fp.Fp`

Represents the prime field `GF(p)`.

#### Construction

- `Fp(p)`:
  Construct the field with `p` prime. Fails if `p` is not prime.

#### Accessors and constructors

- `modulus()`:
  Return the prime modulus `p`.
- `zero()`:
  Return the additive identity.
- `one()`:
  Return the multiplicative identity.
- `elem(k)`:
  Convert an integer to a field element modulo `p`.
- `elements()`:
  Enumerate all elements of the field.
- `nonzero_elements()`:
  Enumerate all nonzero elements.

#### Field operations

- `add(a, b)`:
  Return `a + b`.
- `neg(a)`:
  Return `-a`.
- `sub(a, b)`:
  Return `a - b`.
- `mul(a, b)`:
  Return `a * b`.
- `inv(a)`:
  Return the multiplicative inverse of `a`. Fails on `0`.
- `pow(a, e)`:
  Return `a^e`. Negative exponents require `a != 0`.

#### Number-theoretic utilities

- `mul_order(a)`:
  Return the multiplicative order of nonzero `a` in `GF(p)^*`.
- `is_abelian_mul()`:
  Return `True`. This is a sanity helper for the multiplicative structure.
- `is_quadratic_residue(a)`:
  Return whether `a` is a quadratic residue, using Euler's criterion. Returns `True` for `0`.
- `sqrt(a)`:
  Return one square root of `a` if one exists, else `None`. Uses brute force, so it is suitable for small primes.
- `dlog(g, h)`:
  Solve `g^x = h` in the multiplicative group using baby-step giant-step. Returns the smallest nonnegative solution when one exists.

### `field.fp.FpElem`

Represents an element of `GF(p)`.

#### Element methods

- `value()`:
  Return the canonical integer representative.
- `mul_order()`:
  Return the multiplicative order of the element. Fails on `0`.
- `inv()`:
  Return the multiplicative inverse. Fails on `0`.

#### Python protocol methods

- `__repr__()`:
  Return a string such as `3 (mod 7)`.
- `__int__()`:
  Convert to the canonical integer representative.
- `__richcmp__(other, op)`:
  Support `==` and `!=` for elements from the same field.
- `__neg__()`:
  Return the additive inverse.
- `__add__(other)`:
  Support field addition.
- `__sub__(other)`:
  Support field subtraction.
- `__mul__(other)`:
  Support field multiplication.
- `__truediv__(other)`:
  Divide by `other`. Fails if `other` is zero.
- `__pow__(exp, modulo=None)`:
  Support exponentiation with integer exponents. Three-argument `pow` is rejected.
- `__radd__(other)`:
  Support `int + FpElem`.
- `__rsub__(other)`:
  Support `int - FpElem`.
- `__rmul__(other)`:
  Support `int * FpElem`.
- `__rtruediv__(other)`:
  Support `int / FpElem` inside `GF(p)`.

### `field.poly_fp.PolyFp`

Represents a polynomial over `F_p` stored as coefficients from low degree to high degree.

#### Construction

- `PolyFp(p, coeffs)`:
  Construct a polynomial over `F_p`. Coefficients are given low degree to high degree, canonicalized modulo `p`, and trimmed of trailing zeros.

#### Accessors and predicates

- `p()`:
  Return the base field characteristic.
- `coeffs()`:
  Return the coefficient list.
- `degree()`:
  Return the polynomial degree, or `-1` for the zero polynomial.
- `is_zero()`:
  Return whether the polynomial is zero.
- `is_one()`:
  Return whether the polynomial is the constant polynomial `1`.

#### Polynomial operations

- `monic()`:
  Scale the polynomial so its leading coefficient becomes `1`. Fails for the zero polynomial.
- `add(other)`:
  Return the sum.
- `neg()`:
  Return the additive inverse.
- `sub(other)`:
  Return the difference.
- `mul(other)`:
  Return the product.
- `scale(k)`:
  Multiply the polynomial by the scalar `k` modulo `p`.
- `div_rem(modulus)`:
  Perform polynomial long division and return `(quotient, remainder)`.
- `modulo(modulus)`:
  Return the remainder after division by `modulus`.
- `gcd(other)`:
  Return the monic greatest common divisor.
- `egcd(other)`:
  Return `(g, s, t)` satisfying `g = s*self + t*other`.
- `is_irreducible()`:
  Return whether the polynomial is irreducible over `F_p`. Uses brute force and is intended for small degrees.

#### Python protocol methods

- `__repr__()`:
  Return a compact debug-style representation.

### `field.fq.Fq`

Represents an extension field `GF(p^k)` as `F_p[x] / (f)` where `f` is the supplied modulus polynomial.

#### Construction

- `Fq(p, modulus_coeffs)`:
  Construct the extension field using the polynomial with coefficients `modulus_coeffs`, listed low degree to high degree. The implementation requires the modulus to have degree at least `1` and to be monic. Intended use assumes the modulus is irreducible.

#### Accessors and constructors

- `p()`:
  Return the base characteristic.
- `degree()`:
  Return the extension degree `k`.
- `size()`:
  Return the field size `p^k`.
- `modulus_coeffs()`:
  Return the modulus polynomial coefficients.
- `elem(coeffs)`:
  Construct a field element from polynomial coefficients listed low degree to high degree, then reduce it modulo the field modulus.
- `zero()`:
  Return the additive identity.
- `one()`:
  Return the multiplicative identity.
- `elements(max_size=None)`:
  Enumerate all field elements if the field size is within the configured bound.

#### Field operations

- `add(a, b)`:
  Return `a + b`.
- `neg(a)`:
  Return `-a`.
- `sub(a, b)`:
  Return `a - b`.
- `mul(a, b)`:
  Return `a * b`, reduced modulo the field modulus.
- `inv(a)`:
  Return the multiplicative inverse using polynomial extended GCD. Fails on `0`.
- `pow(a, exp)`:
  Return `a^exp`. Negative exponents use inversion.
- `mul_order(a)`:
  Return the multiplicative order of nonzero `a`.

#### Enumeration utilities

- `primitive_elements()`:
  Return a bounded list of primitive elements, using default enumeration limits.
- `primitive_elements_with_limit(max_size, limit)`:
  Return up to `limit` primitive elements, failing if the field is too large to enumerate under `max_size`.

### `field.fq.FqElem`

Represents an element of a fixed extension field.

#### Element methods

- `coeffs()`:
  Return the reduced coefficient vector.
- `is_zero()`:
  Return whether the element is zero.
- `trace()`:
  Return the field trace down to the prime subfield, computed as `sum(a^(p^i))`.
- `norm()`:
  Return the field norm down to the prime subfield, computed as `prod(a^(p^i))`.
- `inv()`:
  Return the multiplicative inverse. Fails on `0`.

#### Python protocol methods

- `__bool__()`:
  Treat nonzero elements as truthy.
- `__repr__()`:
  Return a polynomial-style string such as `1 + x^2 in GF(2^3)`.
- `__richcmp__(other, op)`:
  Support `==` and `!=` for elements from the same extension field.
- `__neg__()`:
  Return the additive inverse.
- `__add__(other)`:
  Support field addition.
- `__sub__(other)`:
  Support field subtraction.
- `__mul__(other)`:
  Support field multiplication with reduction modulo the field modulus.
- `__truediv__(other)`:
  Divide by `other` via inversion.
- `__pow__(exp, modulo=None)`:
  Support exponentiation with integer exponents. Three-argument `pow` is rejected.
- `__radd__(other)`:
  Support `int + FqElem`, embedding the integer into the prime subfield.
- `__rsub__(other)`:
  Support `int - FqElem`.
- `__rmul__(other)`:
  Support `int * FqElem`.
- `__rtruediv__(other)`:
  Support `int / FqElem`.

## Notes

- Parent-object methods such as `Fp.add(a, b)` and `Zn.mul(a, b)` validate that the arguments come from the same algebraic structure.
- Element objects expose Python operator overloads so interactive use can usually prefer `a + b`, `a * b`, `a / b`, and `a ** e`.
- Enumeration helpers in `Sn` and `Fq` are intentionally guarded by size limits to avoid accidental blow-ups.
